FetchApiDataService Documentation

Service for handling user registration operations.

This service provides methods for registering users, user login, retrieving movies, adding movies to favorites, deleting favorite movies, and deleting user accounts.

postUserRegistration(userDetails)

Registers a new user.

  • userDetails - The details of the user to register.
  • userDetails.Username - The username of the user.
  • userDetails.Email - The email of the user.
  • userDetails.Password - The password of the user.
  • userDetails.Birthday - the birthday of the user.

Returns an observable that emits the HTTP response data upon success or an error object if the request fails.

Example:

userService.postUserRegistration(userDetails).subscribe(
			(response) => console.log('Registration successful:', response),
			(error) => console.error('Registration failed:', error)
		);

ENDPOINT: https://history-movie-api.onrender.com/register

postUserLogin(userDetails)

Logs in a user.

  • userDetails - The details of the user to login.
  • userDetails.Username - The username of the user.
  • userDetails.Password - The password of the user.

Returns an observable that emits the HTTP response data upon success or an error object if the request fails.

ENDPOINT: https://history-movie-api.onrender.com/login

getAllMovies()

Retrieves a list of Movies from the API.

Returns an observable that emits the HTTP response data upon success or an error object if the request fails.

ENDPOINT: https://history-movie-api.onrender.com/movies

postMovieToFavorite(userID, favoriteMovieID)

Favorites selected movie to your user account.

  • userID - the user id used to find where to post the favorite movie.
  • favoriteMovieID - the favorite movie id to insert into the userDetails.

Returns an observable that emits the HTTP response data upon success or an error object if the request fails.

deleteFavoriteMovie(userID, movieID)

Deletes favorite movie from user's profile.

  • userID - the user id.
  • movieID - the movie id of the favorite movie.

Returns an observable that emits the HTTP response data upon success or an error object if the request fails.

deleteUser(userID)

  • userID - The user's ID for deletion.
